# virtual-keyboard
A fully functional, interactive virtual keyboard built with vanilla HTML, CSS, and JavaScript. This project demonstrates a clean, modern on-screen keyboard that supports both mouse clicks and physical keyboard input.
 Features
Click & Type – Use your mouse to click on any virtual key

Physical Keyboard Support – Type using your actual keyboard; the virtual keys update in real-time

Smart Key States – Shift and CapsLock keys toggle and visually indicate their active state

Full Keyboard Layout – Includes letters, numbers, symbols, and special keys (Backspace, Enter, Tab, Space, etc.)

Responsive Design – Adapts seamlessly to desktop, tablet, and mobile screens

Modern UI – Glass-morphism design with smooth animations and visual feedback

No External Dependencies – Pure vanilla JavaScript, no libraries or frameworks required
